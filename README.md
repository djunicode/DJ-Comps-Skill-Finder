# DJ-Comps-Skill-Finder

## Copy .sample.env to .env

```
SECRET_KEY=<SECRET_KEY> from settings.py
DEBUG=TRUE (for developement) | False (for production)
ENGINE=django.db.backends.postgresql (for PostgreSQL) | django.db.backends.mysql (for MySQL)
DB_NAME=<DATABASE_NAME>
DB_USER=<DATABASE_USER>
DB_PASS=<DATABASE_PASSWORD> 
DB_HOST=localhost (for development) 
DB_PORT=5432 (for PostgreSQL) | 3306 (for MySQL)
```
